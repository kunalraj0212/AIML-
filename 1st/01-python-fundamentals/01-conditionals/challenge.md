# Challenge: ATM Withdrawal Validator

## Requirements
- Take balance amount and withdrawal request amount as float inputs.
- Check if withdrawal amount is positive and does not exceed balance.
- Check if withdrawal amount is a multiple of 100 (using `% 100`).
- If all checks pass, deduct from balance and print success with new balance.
- If any checks fail, print the corresponding error reason.

## Restrictions
- ❌ No AI assistants
- ❌ No online tutorials

## Time Limit
- ⏱️ 30 minutes
